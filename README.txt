This folder contains beginner-to-intermediate Java programs that demonstrate core concepts of Java:

1. SimpleCalculator.java
   - A basic calculator using switch-case and user input.

2. MarkSheet.java
   - Accepts 5 subject marks, calculates total, average, and grade.

3. ATMSimulator.java
   - Menu-driven program to check balance, deposit and withdraw money.

4. BankAccount.java / BankApp.java
   - Object-oriented simulation of a bank account system.

5. FileHandlingDemo.java
   - Reads from and writes to a file using Java File I/O classes.
